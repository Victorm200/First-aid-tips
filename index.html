⛑️ Quick First Aid Tips

🫀 CPR

1. Check responsiveness.
2. Call for help.
3. Start chest compressions – 100–120 per minute.
4. Give rescue breaths if trained.

🔥 Burns

1. Cool the burn with running water for 10–20 minutes.
2. Cover with a clean, non-stick cloth.
3. Don’t apply oil or cream.
4. Call for medical help if severe.

🩸 Bleeding

1. Apply pressure with cloth.
2. Keep injured part raised.
3. Don't remove soaked cloth – add more layers.
4. Call emergency services.

😮‍💨 Choking

1. Ask if they can speak.
2. Give 5 back blows between shoulder blades.
3. Follow with 5 abdominal thrusts.
4. Repeat if needed.
